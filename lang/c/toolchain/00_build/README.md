# C 语言项目构建工具

本目录收录 C 语言项目构建相关的工具文档。

## 构建系统概览

| 工具 | 类型 | 说明 |
|------|------|------|
| [CMake](cmake.md) | 元构建系统 | 跨平台构建系统生成器，事实标准 |
| [Make](make.md) | 构建系统 | 传统 Unix 构建工具，配合 Makefile 使用 |
| [Ninja](ninja.md) | 构建系统 | 高速构建系统，CMake 默认后端 |
| [Meson](meson.md) | 元构建系统 | 现代化构建系统，构建速度快 |
| [Bazel](bazel.md) | 构建系统 | Google 构建，支持大规模项目 |
| [xmake](xmake.md) | 构建系统 | 国产构建工具，简单易用 |

## 工具选型建议

| 场景 | 推荐工具 | 理由 |
|------|----------|------|
| 跨平台项目 | CMake + Ninja | 生态最完善，IDE 支持好 |
| 小型项目 | Make | 简单直接，无依赖 |
| 大型项目 | Bazel | 增量构建，依赖管理强 |
| 追求速度 | Meson / Ninja | 构建速度最快 |
| 国内开发 | xmake | 文档中文友好，学习成本低 |

## 构建流程

```
源代码 (.c/.h)
      ↓
   预处理器
      ↓
    编译器
      ↓
    汇编器
      ↓
    链接器
      ↓
可执行文件/库
```
