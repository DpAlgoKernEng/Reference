# C 语言包管理器

本目录收录 C 语言包管理工具文档。

## 包管理器概览

| 工具 | 说明 | 仓库规模 |
|------|------|----------|
| [vcpkg](vcpkg.md) | 微软 C++ 包管理器 | 2000+ 库 |
| [Conan](conan.md) | 跨平台 C/C++ 包管理器 | 1500+ 库 |
| [CPM](cpm.md) | CMake 原生包管理 | Git 仓库 |

## 为什么需要包管理器

传统方式管理依赖的问题：

| 问题 | 传统方式 | 包管理器 |
|------|----------|----------|
| 依赖查找 | 手动下载 | 自动获取 |
| 版本管理 | 手动记录 | 自动管理 |
| 编译集成 | 手动配置 | 自动集成 |
| 依赖传递 | 手动处理 | 自动解析 |

## 工具选型建议

| 场景 | 推荐工具 | 理由 |
|------|----------|------|
| Windows 开发 | vcpkg | 微软生态，MSVC 集成好 |
| 跨平台项目 | Conan | 灵活性高，支持多种构建系统 |
| CMake 项目 | CPM | 无需额外工具，CMake 原生 |
| CI/CD 集成 | Conan | 缓存机制完善 |

## 基本使用流程

```
声明依赖 (package.json/conanfile.txt)
        ↓
    安装依赖
        ↓
   集成构建系统
        ↓
     编译项目
```
