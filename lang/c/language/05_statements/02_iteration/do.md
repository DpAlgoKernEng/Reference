# do-while 循环 (do-while loop)

## 1. 概述 (Overview)

**do-while 循环**是一种先执行后判断的循环结构，循环体至少执行一次，然后根据条件决定是否继续。

### 核心概念

- **先执行后判断**：循环体先执行，再检查条件
- **至少执行一次**：无论条件如何，循环体至少执行一次
- **后置条件检查**：条件在每次迭代后求值

### 技术定位

do-while 循环属于 C 语言的**迭代语句**（iteration statements）类别，与 while 和 for 循环共同构成循环控制结构。其独特之处在于保证循环体至少执行一次。

### 历史背景

do-while 循环的设计动机是处理"至少执行一次"的场景，如输入验证、菜单选择、数字转换等。

## 3. 语法与参数 (Syntax and Parameters)

### 语法形式

```c
attr-spec-seq(可选) do statement while ( expression ) ;
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `attr-spec-seq` | (C23 起) 可选的属性列表，应用于循环语句 |
| `statement` | 任意语句（通常是复合语句），作为循环体 |
| `expression` | 任意标量类型表达式，每次迭代后求值，等于零时退出 |

### 表达式求值规则

- **类型**：任意标量类型（整数、浮点、指针等）
- **求值时机**：每次循环体执行后
- **退出条件**：表达式等于零时退出循环

## 4. 底层原理 (Underlying Principles)

### 执行流程

```
┌─────────────────────────────────────┐
│         进入 do-while 循环          │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│         执行 statement（循环体）      │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│         求值 expression              │
└─────────────────┬───────────────────┘
                  │
        ┌─────────┴─────────┐
        │                   │
        ▼                   ▼
┌───────────────┐   ┌───────────────┐
│ 非零（继续）   │   │ 零（退出）     │
└───────┬───────┘   └───────────────┘
        │
        └──────► 返回执行循环体
```

### 与 while 循环的关键区别

| 特性 | do-while | while |
|------|----------|-------|
| 条件检查时机 | 迭代后 | 迭代前 |
| 最少执行次数 | 至少 1 次 | 可能 0 次 |
| 适用场景 | 需要先执行后判断 | 先检查再执行 |

### goto 进入循环体的行为

如果通过 goto 进入循环体中间：
- 循环体正常执行
- 表达式在循环体执行后求值
- 循环可能正常继续

### 无限循环行为

无限循环如果没有任何可观察行为（I/O、volatile 访问、原子或同步操作），则具有未定义行为。例外：条件为常量表达式的循环 `do {...} while(true);` 总是无限循环。

### 块作用域（C99 起）

do-while 语句建立独立的块作用域：表达式中引入的标识符在语句结束后超出作用域。

## 5. 使用场景 (Use Cases)

### 适用场景

| 场景 | 说明 |
|------|------|
| 输入验证 | 先读取输入，再判断是否有效 |
| 菜单选择 | 先显示菜单，再处理选择 |
| 数字转换 | 至少产生一位数字 |
| 资源清理 | 确保清理操作执行 |

### 最佳实践

1. **使用复合语句**：循环体使用花括号，提高可读性
2. **明确终止条件**：确保循环最终能够终止
3. **配合 break/continue**：灵活控制循环流程

### 常见陷阱

| 陷阱 | 说明 | 解决方案 |
|------|------|----------|
| 忘记分号 | `while(condition)` 后缺少分号 | 记得添加 `;` |
| 无限循环 | 条件始终为真 | 确保条件能变为假 |
| 分号位置 | 分号放在错误位置 | 分号在 `while(condition)` 之后 |

## 6. 代码示例 (Examples)

### 基础用法

```c
#include <stdio.h>
#include <stdlib.h>

enum { SIZE = 8 };

int main(void)
{
    // 简单示例
    int array[SIZE], n = 0;

    do array[n++] = rand() % 2;  // 循环体是单条表达式语句
    while(n < SIZE);

    puts("Array filled!");
    n = 0;

    do
    {  // 循环体是复合语句
        printf("%d ", array[n]);
        ++n;
    } while (n < SIZE);

    printf("\n");

    return 0;
}
```

**输出：**
```
Array filled!
1 0 1 1 1 1 0 0
```

### 数字转换（K&R itoa 示例）

```c
#include <stdio.h>

// do-while 用于确保至少产生一位数字
int main(void)
{
    int num = 1234, i = 0;
    char s[10];

    do s[i++] = num % 10 + '0';  // 获取下一位数字（逆序）
    while ((num /= 10) > 0);

    s[i] = '\0';
    puts(s);  // 输出: 4321

    return 0;
}
```

### 输入验证

```c
#include <stdio.h>

int main(void)
{
    int choice;

    do
    {
        printf("Enter a number (1-5): ");
        scanf("%d", &choice);

        if (choice < 1 || choice > 5)
            printf("Invalid input! Try again.\n");

    } while (choice < 1 || choice > 5);

    printf("You chose: %d\n", choice);

    return 0;
}
```

### 使用 break 和 continue

```c
#include <stdio.h>

int main(void)
{
    int i = 0;

    do
    {
        i++;
        if (i == 3)
            continue;  // 跳过打印 3
        if (i == 7)
            break;     // 在 7 处退出
        printf("%d ", i);
    } while (i < 10);

    printf("\n");
    return 0;
}
```

**输出：**
```
1 2 4 5 6
```

### 与 while 循环对比

```c
#include <stdio.h>

int main(void)
{
    // while 循环：条件不满足时一次都不执行
    int x = 10;
    while (x < 5)
    {
        printf("while: x = %d\n", x);  // 不执行
        x++;
    }

    // do-while 循环：至少执行一次
    x = 10;
    do
    {
        printf("do-while: x = %d\n", x);  // 执行一次
        x++;
    } while (x < 5);

    return 0;
}
```

**输出：**
```
do-while: x = 10
```

## 7. 总结 (Summary)

### 核心要点

| 特性 | 说明 |
|------|------|
| 执行顺序 | 先执行循环体，后检查条件 |
| 最少执行次数 | 至少 1 次 |
| 条件类型 | 任意标量类型 |
| 语法特点 | 必须以分号结尾 |

### 三种循环对比

| 特性 | while | do-while | for |
|------|-------|----------|-----|
| 条件检查 | 前 | 后 | 前 |
| 最少执行 | 0 次 | 1 次 | 0 次 |
| 初始化 | 手动 | 手动 | 内置 |
| 适用场景 | 条件驱动 | 至少执行一次 | 计数驱动 |

### 学习建议

1. **理解执行顺序**：先执行后判断是关键区别
2. **选择合适循环**：需要至少执行一次时使用 do-while
3. **注意分号**：不要忘记 `while(condition);` 后的分号
4. **经典场景**：输入验证、数字转换、菜单选择
