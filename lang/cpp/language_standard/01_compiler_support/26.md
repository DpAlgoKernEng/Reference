# C++26 编译器支持

## 1. 概述

C++26 是 C++ 语言的下一个主要标准版本，计划于 2026 年正式发布。本文档追踪各大编译器对 C++26 核心语言特性和库特性的支持情况。

### 编译器支持状态说明

| 符号 | 含义 |
|------|------|
| Yes | 已完整支持 |
| 版本号 | 从该版本开始支持 |
| 空白 | 尚未支持 |
| * | 需要查看补充说明 |
| DR**nn** | 缺陷报告，**nn** 表示目标 C++ 版本，如 DR20 表示应用于 C++20 |

### 主要编译器

| 编译器 | 说明 |
|--------|------|
| GCC | GNU Compiler Collection |
| Clang | LLVM 编译器前端 |
| MSVC | Microsoft Visual C++ |
| Apple Clang | Apple 版 Clang |
| EDG eccp | Edison Design Group 编译器 |
| Intel C++ | Intel C++ 编译器 |
| Nvidia HPC C++ | Nvidia 高性能计算编译器（原 PGI） |
| Nvidia nvcc | Nvidia CUDA 编译器 |
| Cray | Cray 编译器 |

## 2. 来源与演变

### 标准化进程

C++26 标准化工作由 ISO C++ 委员会（WG21）主导，采用提案驱动的方式：

1. **提案阶段**：特性以论文（Paper）形式提交
2. **评审阶段**：经过多次会议讨论和修订
3. **投票阶段**：最终投票纳入标准
4. **实现阶段**：编译器厂商实现支持

### 版本历史

| 版本 | 发布年份 | 主要特性 |
|------|----------|----------|
| C++98 | 1998 | 首个 ISO 标准 |
| C++11 | 2011 | 现代化革命性更新 |
| C++14 | 2014 | 小幅改进 |
| C++17 | 2017 | 结构化绑定、optional 等 |
| C++20 | 2020 | 概念、范围、模块、协程 |
| C++23 | 2023 | 标准库增强 |
| C++26 | 2026（计划） | 静态反射、契约等 |

## 6. 使用场景与建议

### 开发环境选择

**生产环境推荐**：
- 优先使用稳定版本编译器
- 谨慎采用 C++26 特性，需验证编译器支持

**实验性开发**：
- 使用 GCC 15 / Clang 19+ 获取最新支持
- MSVC 19.40+ 支持部分格式化特性

### 特性可用性检查

使用特性测试宏（Feature Test Macro, FTM）检查编译器支持：

```cpp
// 检查参数包索引支持
#if __cpp_pack_indexing >= 202311L
    // 使用参数包索引
#endif

// 检查 constexpr void* 转换
#if __cpp_constexpr_cast_from_void_ptr >= 202306L
    // 使用 constexpr void* 转换
#endif

// 检查结构化绑定属性
#if __cpp_structured_bindings >= 202411L
    // 使用结构化绑定属性
#endif
```

### 编译器标志

```bash
# GCC/Clang
g++ -std=c++26 main.cpp
clang++ -std=c++26 main.cpp

# MSVC
cl /std:c++26 main.cpp
```

### 跨平台兼容性

```cpp
// 使用条件编译处理特性差异
#if defined(__cpp_constexpr_exceptions)
    constexpr void may_throw() {
        throw 42;  // C++26 constexpr 异常
    }
#else
    void may_throw() {
        throw 42;  // 回退到运行时
    }
#endif
```

## 7. 总结

### 核心要点

1. **C++26 仍处于开发中**：标准预计 2026 年正式发布，编译器支持持续更新
2. **GCC 和 Clang 领先**：开源编译器对新特性支持较为积极
3. **MSVC 稳步跟进**：微软编译器在标准库特性上进展较快
4. **特性测试宏重要**：使用 FTM 确保代码可移植性

### 编译器支持现状

| 编译器 | 核心语言支持 | 库支持 | 总体评估 |
|--------|--------------|--------|----------|
| GCC 14/15 | 良好 | 良好 | 积极跟进 |
| Clang 18/19/20 | 良好 | 良好 | 积极跟进 |
| MSVC 19.40+ | 一般 | 良好 | 稳步跟进 |
| Apple Clang | 一般 | 一般 | 依赖上游 |

### 学习建议

1. **关注提案论文**：通过论文编号（如 P2662R3）追踪特性详情
2. **查阅 cppreference**：获取最新的编译器支持信息
3. **实验性使用**：在新项目中尝试新特性，积累经验
4. **生产谨慎**：生产环境需评估编译器支持稳定性
