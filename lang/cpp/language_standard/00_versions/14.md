# C++14 语言标准

## 1. 概述

C++14 是 C++11 之后的次要版本，主要包含小幅改进和缺陷修复。该标准于 2014 年 8 月 18 日宣布批准，2014 年 12 月 15 日正式发布。

C++14 的定位是对 C++11 的"补全"版本，而非引入大量新特性的重大更新。在正式批准前，业界常使用 **C++1y** 作为代号，表示该版本将于 2010 年代发布。

### 版本定位

| 版本 | 类型 | 特点 |
|------|------|------|
| C++11 | 重大版本 | 革命性变化，引入大量现代特性 |
| **C++14** | 次要版本 | 小幅改进，完善 C++11 特性 |
| C++17 | 重大版本 | 引入新特性，合并 TS |

## 2. 来源与演变

### 发布时间线

| 里程碑 | 日期 |
|--------|------|
| 开始开发 | 2012 年 |
| 批准公告 | 2014 年 8 月 18 日 |
| 正式发布 | 2014 年 12 月 15 日 |

### 设计动机

C++14 的主要设计目标：

1. **完善 C++11 特性**：修复 C++11 中遗留的问题和不一致之处
2. **提高易用性**：简化常见编程模式的语法
3. **缺陷修复**：解决 276 个核心缺陷和 158 个库缺陷

### 代号说明

C++14 在开发阶段被称为 **C++1y**，其中 'y' 表示 2010 年代的某一年。类似地：
- C++11 曾被称为 C++0x
- C++17 曾被称为 C++1z

## 3. 新语言特性

### 3.1 变量模板（Variable Templates）

变量模板允许为变量定义模板参数。

```cpp
// 基础用法：定义数学常量
template<typename T>
constexpr T pi = T(3.14159265358979323846);

// 使用
float f = pi<float>;
double d = pi<double>;
long double ld = pi<long double>;

// 特化
template<>
constexpr int pi<int> = 3;
```

### 3.2 泛型 Lambda（Generic Lambdas）

Lambda 参数可以使用 `auto` 声明，实现泛型功能。

```cpp
// C++11：需要显式指定类型
auto add_int = [](int a, int b) { return a + b; };

// C++14：泛型 lambda
auto add = [](auto a, auto b) { return a + b; };

// 用法
add(1, 2);        // int
add(1.5, 2.5);    // double
add(std::string("a"), std::string("b")); // string
```

### 3.3 Lambda 初始化捕获（Lambda Init-Capture）

允许在捕获列表中初始化新变量，实现移动捕获。

```cpp
// 移动捕获
auto ptr = std::make_unique<int>(42);

// C++11：无法移动捕获
// C++14：支持初始化捕获
auto f = [p = std::move(ptr)]() { return *p; };

// 在捕获中创建新变量
auto f2 = [x = 10, y = 20]() { return x + y; };

// 捕获表达式结果
auto f3 = [str = std::to_string(42)]() { return str; };
```

### 3.4 放宽的 constexpr 限制

C++14 大幅放宽了 `constexpr` 函数的限制。

```cpp
// C++14 允许的特性：
// - 局部变量声明
// - if 和 switch 语句
// - 循环语句（for、while、do-while）
// - 多条 return 语句

constexpr int factorial(int n) {
    if (n <= 1) return 1;           // if 语句
    int result = 1;                  // 局部变量
    for (int i = 2; i <= n; ++i) {   // for 循环
        result *= i;
    }
    return result;
}

// 编译期计算
static_assert(factorial(5) == 120, "");
```

### 3.5 二进制字面量（Binary Literals）

使用 `0b` 或 `0B` 前缀表示二进制数字。

```cpp
int binary = 0b1010;        // 10
int mask = 0b11110000;      // 240
int flags = 0b0000'1111;    // 结合数字分隔符

// 常见用途：位掩码
constexpr int read_perm =    0b0001;
constexpr int write_perm =   0b0010;
constexpr int execute_perm =  0b0100;
```

### 3.6 数字分隔符（Digit Separators）

使用单引号 `'` 作为数字分隔符，提高可读性。

```cpp
int million = 1'000'000;
int hex_value = 0xFF'FF'FF;
int binary = 0b1010'1100'0011'1111;
double pi = 3.141'592'653'589'793;

// 分隔符位置任意，不影响值
int a = 1'2'3'4;  // 等价于 1234
int b = 1'000;    // 等价于 1000
```

### 3.7 函数返回类型推导

使用 `auto` 和 `decltype(auto)` 推导函数返回类型。

```cpp
// auto 返回类型推导
auto add(int a, int b) {
    return a + b;  // 推导为 int
}

// decltype(auto) 保持引用和 cv 限定符
decltype(auto) get_ref(int& x) {
    return x;  // 返回 int&
}

decltype(auto) get_value(int x) {
    return x;  // 返回 int
}

// 递归函数也支持
auto factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```

### 3.8 聚合类与默认成员初始化器

聚合类可以包含默认成员初始化器。

```cpp
struct Point {
    int x = 0;
    int y = 0;
    int z = 0;
};

// 聚合初始化
Point p1{};           // {0, 0, 0}
Point p2{1, 2};       // {1, 2, 0}
Point p3{1, 2, 3};    // {1, 2, 3}

// 注意：如果有用户定义的构造函数，则不是聚合类
struct NonAggregate {
    int x = 0;
    int y = 0;
    NonAggregate() = default;  // C++14 仍允许作为聚合类
};
```

### 3.9 `[[deprecated]]` 属性

标记已废弃的实体，编译器会发出警告。

```cpp
[[deprecated]] void old_function();

[[deprecated("Use new_function() instead")]]
void legacy_function();

class [[deprecated]] OldClass {};

// 使用时会收到警告
old_function();  // warning: 'old_function' is deprecated
```

### 3.10 sized deallocation

新增带大小的 `delete` 运算符重载。

```cpp
// 自定义带大小的删除函数
void operator delete(void* ptr, std::size_t size) noexcept;

// 使用场景：内存池优化
class MemoryPool {
public:
    void* allocate(std::size_t size);
    void deallocate(void* ptr, std::size_t size);
};

// 编译器可自动调用带大小的版本
void operator delete(void* ptr, std::size_t size) noexcept {
    MemoryPool::deallocate(ptr, size);
}
```

## 4. 新库特性

### 4.1 std::make_unique

创建 `std::unique_ptr` 的工厂函数。

```cpp
#include <memory>

// 创建 unique_ptr
auto p1 = std::make_unique<int>(42);
auto p2 = std::make_unique<std::string>("Hello");
auto p3 = std::make_unique<std::vector<int>>(10, 5);

// 创建数组
auto arr = std::make_unique<int[]>(10);  // 10 个零初始化的 int

// 对比 make_shared
auto sp = std::make_shared<int>(42);  // C++11 已有
```

### 4.2 std::shared_timed_mutex 和 std::shared_lock

读写锁支持，允许多读单写。

```cpp
#include <shared_mutex>
#include <mutex>

class ThreadSafeCache {
    mutable std::shared_timed_mutex mutex_;
    std::map<std::string, int> data_;

public:
    // 读操作：共享锁
    int get(const std::string& key) const {
        std::shared_lock<std::shared_timed_mutex> lock(mutex_);
        auto it = data_.find(key);
        return it != data_.end() ? it->second : -1;
    }

    // 写操作：独占锁
    void set(const std::string& key, int value) {
        std::unique_lock<std::shared_timed_mutex> lock(mutex_);
        data_[key] = value;
    }
};
```

### 4.3 std::integer_sequence

编译期整数序列，用于元编程。

```cpp
#include <utility>
#include <tuple>

// 基本用法
using seq = std::integer_sequence<int, 0, 1, 2, 3, 4>;
using index_seq = std::index_sequence<0, 1, 2, 3, 4>;

// 辅助类型
using seq5 = std::make_integer_sequence<int, 5>;  // 0,1,2,3,4
using idx5 = std::make_index_sequence<5>;          // size_t: 0,1,2,3,4

// 实际应用：解包 tuple
template<typename... Args, std::size_t... Is>
void print_tuple(const std::tuple<Args...>& t, 
                 std::index_sequence<Is...>) {
    ((std::cout << std::get<Is>(t) << " "), ...);
}

template<typename... Args>
void print_tuple(const std::tuple<Args...>& t) {
    print_tuple(t, std::make_index_sequence<sizeof...(Args)>{});
}
```

### 4.4 std::exchange

原子交换并返回旧值。

```cpp
#include <utility>

// 基本用法
int x = 10;
int old = std::exchange(x, 20);  // x = 20, old = 10

// 典型用途：移动语义
class Buffer {
    std::vector<int> data_;
public:
    std::vector<int> take_data() {
        return std::exchange(data_, {});  // 移动并清空
    }
};

// 实现移动赋值运算符
class Resource {
    int* ptr_;
public:
    Resource& operator=(Resource&& other) noexcept {
        delete std::exchange(ptr_, std::exchange(other.ptr_, nullptr));
        return *this;
    }
};
```

### 4.5 std::quoted

字符串引用包装器，保留空格和引号。

```cpp
#include <iomanip>
#include <sstream>

// 输出带引号的字符串
std::string s = "Hello World";
std::cout << std::quoted(s);  // 输出: "Hello World"

// 读取带引号的字符串
std::istringstream iss(R"("Hello World" "Another String")");
std::string a, b;
iss >> std::quoted(a) >> std::quoted(b);
// a = "Hello World", b = "Another String"

// 用途：CSV 处理
std::ostringstream oss;
oss << std::quoted("value with, comma") << ",";
// 输出: "value with, comma",
```

### 4.6 异构关联查找（Heterogeneous Lookup）

关联容器支持使用不同类型查找，无需构造临时 key 对象。

```cpp
#include <map>
#include <string>

struct StringComparator {
    using is_transparent = void;
    
    bool operator()(const std::string& lhs, const std::string& rhs) const {
        return lhs < rhs;
    }
    
    bool operator()(const std::string& lhs, std::string_view rhs) const {
        return lhs < rhs;
    }
    
    bool operator()(std::string_view lhs, const std::string& rhs) const {
        return lhs < rhs;
    }
};

std::map<std::string, int, StringComparator> m;
m["key"] = 42;

// 直接使用 string_view 查找，无需构造临时 string
std::string_view key = "key";
auto it = m.find(key);  // C++14 支持
```

### 4.7 按类型访问 tuple

`std::get` 支持按类型访问 tuple 元素。

```cpp
#include <tuple>
#include <string>

std::tuple<int, std::string, double> t{42, "hello", 3.14};

// C++11：按索引访问
int i = std::get<0>(t);

// C++14：按类型访问
std::string s = std::get<std::string>(t);
double d = std::get<double>(t);

// 注意：如果类型重复则编译错误
std::tuple<int, int, double> t2{1, 2, 3.0};
// std::get<int>(t2);  // 错误：int 出现两次
```

### 4.8 双范围算法

部分算法支持双范围版本。

```cpp
#include <algorithm>
#include <vector>

std::vector<int> v1{1, 2, 3, 4, 5};
std::vector<int> v2{1, 2, 3, 4, 5, 6};
int arr[] = {1, 2, 3, 4, 5};

// C++11：需要手动指定范围
bool eq1 = std::equal(v1.begin(), v1.end(), v2.begin());

// C++14：双范围版本
bool eq2 = std::equal(v1.begin(), v1.end(), v2.begin(), v2.end());

// 其他双范围算法
bool perm = std::is_permutation(v1.begin(), v1.end(), 
                                 v2.begin(), v2.end());

auto [mismatch_first, mismatch_second] = 
    std::mismatch(v1.begin(), v1.end(), v2.begin(), v2.end());
```

### 4.9 类型别名版本的类型特征

`_t` 后缀的类型别名，简化类型特征使用。

```cpp
#include <type_traits>

// C++11：需要 typename ...::type
template<typename T>
using remove_ptr_t = typename std::remove_pointer<T>::type;

// C++14：直接使用 _t 后缀
template<typename T>
using remove_ptr_t2 = std::remove_pointer_t<T>;

// 常用别名
static_assert(std::is_integral_t<int>, "");
static_assert(std::is_pointer_t<int*>, "");
static_assert(std::is_array_t<int[]>, "");

// _v 后缀（C++17 才有，但部分编译器在 C++14 已支持）
constexpr bool is_int = std::is_integral<int>::value;  // C++11
constexpr bool is_int_v = std::is_integral_v<int>;     // C++17
```

### 4.10 用户定义字面量扩展

标准库新增时间、字符串和复数的用户定义字面量。

```cpp
#include <chrono>
#include <string>
#include <complex>

using namespace std::chrono_literals;
using namespace std::string_literals;
using namespace std::complex_literals;

// 时间字面量
auto d1 = 10s;     // 10 秒
auto d2 = 100ms;   // 100 毫秒
auto d3 = 500us;   // 500 微秒
auto d4 = 1000ns;  // 1000 纳秒
auto d5 = 1h;      // 1 小时
auto d6 = 30min;  // 30 分钟

// 字符串字面量
auto s1 = "hello"s;  // std::string
auto s2 = L"wide"s;  // std::wstring
auto s3 = u8"utf8"s; // std::string (C++20 后为 std::u8string)

// 复数字面量
auto c1 = 1.0if;   // std::complex<float>(0, 1.0f)
auto c2 = 2.5i;    // std::complex<double>(0, 2.5)
```

## 6. 代码示例

### 6.1 综合示例：泛型工厂

```cpp
#include <memory>
#include <utility>
#include <tuple>

// 使用 C++14 特性实现泛型工厂
template<typename T, typename... Args>
auto make(Args&&... args) {
    return std::make_unique<T>(std::forward<Args>(args)...);
}

// 使用泛型 lambda 处理多种类型
auto process = [](auto& container, auto value) {
    container.push_back(value);
};

// 使用示例
int main() {
    // make_unique
    auto ptr = std::make_unique<int>(42);
    
    // 泛型 lambda
    std::vector<int> vi;
    std::vector<std::string> vs;
    
    process(vi, 10);
    process(vs, "hello"s);
    
    return 0;
}
```

### 6.2 综合示例：线程安全缓存

```cpp
#include <map>
#include <shared_mutex>
#include <string>
#include <memory>

template<typename K, typename V>
class ThreadSafeCache {
    mutable std::shared_timed_mutex mutex_;
    std::map<K, std::shared_ptr<V>> data_;

public:
    std::shared_ptr<V> get(const K& key) const {
        std::shared_lock<std::shared_timed_mutex> lock(mutex_);
        auto it = data_.find(key);
        if (it != data_.end()) {
            return it->second;
        }
        return nullptr;
    }

    void set(const K& key, std::shared_ptr<V> value) {
        std::unique_lock<std::shared_timed_mutex> lock(mutex_);
        data_[key] = std::move(value);
    }

    bool remove(const K& key) {
        std::unique_lock<std::shared_timed_mutex> lock(mutex_);
        return data_.erase(key) > 0;
    }
};
```

### 6.3 综合示例：编译期计算

```cpp
#include <utility>

// C++14 constexpr 实现编译期计算
constexpr int fibonacci(int n) {
    if (n <= 1) return n;
    
    int a = 0, b = 1;
    for (int i = 2; i <= n; ++i) {
        int temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}

// 编译期验证
static_assert(fibonacci(10) == 55, "");

// 变量模板
template<typename T>
constexpr T epsilon = T(0);

template<>
constexpr float epsilon<float> = 1e-6f;

template<>
constexpr double epsilon<double> = 1e-15;

// 使用
constexpr float tol = epsilon<float>;
```

## 7. 总结

### 核心要点

| 类别 | 特性 | 说明 |
|------|------|------|
| **模板** | 变量模板 | 参数化的变量定义 |
| **Lambda** | 泛型参数、初始化捕获 | 更灵活的 Lambda 表达式 |
| **constexpr** | 放宽限制 | 局部变量、循环、条件语句 |
| **字面量** | 二进制、数字分隔符 | 提高代码可读性 |
| **类型推导** | 返回类型推导 | `auto` 和 `decltype(auto)` |
| **属性** | `[[deprecated]]` | 标记废弃 API |
| **智能指针** | `make_unique` | 补齐 C++11 遗漏 |
| **并发** | `shared_timed_mutex` | 读写锁支持 |

### 版本演进

```
C++11 → C++14 → C++17 → C++20
  │        │        │        │
  │        │        │        └─ 概念、协程、模块
  │        │        └─ 结构化绑定、optional、if constexpr
  │        └─ 完善特性、小幅改进
  └─ 现代化基础
```

### 学习建议

1. **优先掌握**：泛型 Lambda、`make_unique`、返回类型推导
2. **适度了解**：变量模板、`integer_sequence`、异构查找
3. **按需学习**：`constexpr` 扩展、读写锁高级用法

### 迁移建议

从 C++11 迁移到 C++14 的建议：

1. 使用 `std::make_unique` 替代 `std::unique_ptr<T>(new T)`
2. 使用泛型 Lambda 简化模板代码
3. 使用返回类型推导简化函数签名
4. 使用 `[[deprecated]]` 标记旧接口
